# Minedit Website

Landing page for [Minedit](https://github.com/Angais/minedit), an open-source NeoForge mod that lets AI models build and edit Minecraft structures.

## Stack

- Vite + React + TypeScript
- Tailwind CSS v4
- Motion (Framer Motion) for scroll and entrance animations
- Official X embeds for the showcase posts

## Development

```sh
npm install
npm run dev
```

## Production build

```sh
npm run build
npm run preview
```

Static output goes to `dist/` — deployable to any static host (GitHub Pages, Vercel, Netlify, Cloudflare Pages).
